// FAQ

What do I need to use these?

Claude Code (Anthropic's CLI), Cursor, Windsurf, or any MCP-compatible AI coding tool. Elara Core installed via pip. Python 3.10+.

Is Elara Core free?

Yes. Elara Core is open source (BSL-1.1) and always will be. These packs are pre-configured, optimized setups built on top of it — saving you hours of configuration.

What format are the files?

ZIP download containing markdown files (CLAUDE.md, .cursorrules), JSON configs, and a setup guide. Drop them into your project and go.

Can I get a refund?

Yes, within 14 days of purchase if the product doesn't work as described. See our refund policy.

Do I get updates?

Yes. When Elara Core releases a new version, we update the packs. You get free access to all future versions of your purchased tier.
